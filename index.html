LSP Downloader Introduction

Welcome to the LSP Downloader! This tool allows you to easily download and manage library code for your SPIKE hub.

How It Works

The LSP Downloader uses a function called exportProgram to save library code to your SPIKE hub. Here's the code that makes it happen:


def exportProgram(code, module): # Function to export the library code string
    import os
    os.listdir("/flash")
    os.chdir('/flash') # change directory to root
    try:
        os.remove(module+".py") # remove any existing library file of the same name
    except:
        pass
    f = open(module+".py", 'w+') # Create a new file customlib.py in the SPIKE hub root
    f.write(code) # Write out the library code string to the customlib.py file
    f.close()

Features

  • Automatically removes existing library files with the same name
  • Creates a new file in the SPIKE hub root directory
  • Writes the library code to the newly created file
